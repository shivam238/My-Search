# MySearch

MySearch is an advanced web app offering fast text and voice search with AI-powered overviews. It features a premium, dynamic UI with light and dark themes. Integrated with Firebase, it provides a secure admin panel and real-time user feedback system, ensuring a safe, fast, and highly engaging user experience.

## Features

- **Advanced Search:** Fast text and voice search capabilities.
- **AI Overviews:** Get quick AI-powered summaries for your queries.
- **Dynamic UI:** Premium user interface with glassmorphism effects and light/dark theme support.
- **Spotify-Style Browse Grid:** Discover trending content easily.
- **Secure Admin Panel:** Protected by Firebase Authentication.
- **Real-time Feedback:** User suggestions and feedback synced securely via Firebase Realtime Database.

## Technologies Used

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend/Database:** Firebase (Authentication & Realtime Database)
- **Local Server:** Python (Simple HTTP Server)

## Getting Started

### Prerequisites

- Python 3.x installed on your system.
- Firebase project setup with Authentication and Realtime Database enabled.

### Installation

1. Clone this repository or download the source code.
2. (Optional) Ensure your Firebase configuration is correctly added in the `index.html` or related scripts.
3. Open a terminal in the project directory.
4. Run the local development server:

   ```bash
   python3 server.py
   ```

5. Open your browser and navigate to `http://localhost:5000`.
